Script Writing Framework (Hook-Optimized Structure)
Every script must follow this structure to maximize TikTok performance and allow deterministic generation/posting.

Script Structure (for 15–60 second video)
Segment 0: Hook (first 2–3 seconds, spoken or text)
Purpose: Stop the scroll. Make the user stay.

Formula: Start with the most surprising/emotional/provocative element of the concept.

Examples:

"This recall affects 100,000 vehicles—and it could be yours."

"A major company just admitted they failed on safety."

"Your car might have a defect that nobody's talking about."

Execution: Speak or display as text overlay immediately. No intro, no setup.

Segment 1: Context (3–10 seconds, spoken or text + visuals)
Purpose: Briefly ground the hook in facts. Explain what and who.

Content: 1–2 key facts from the source article (company name, product, what happened, when).

Execution: B-roll of product/location, on-screen text labels, short sentences.

Segment 2: Why It Matters (5–15 seconds, spoken or text + visuals)
Purpose: Explain why this matters to the viewer (personal stakes, broader impact, emotional relevance).

Content: Impact statement (safety risk, financial consequence, lifestyle change, accountability).

Execution: Visuals that show impact (affected users, damaged product, affected numbers), empathetic narration.

Segment 3: Call-to-Action / Closure (3–5 seconds, text or narration)
Purpose: End with a clear next step or engagement prompt.

Formula: Ask a question, encourage comment, suggest next action, or build curiosity.

Examples:

"Do you own one of these vehicles? Let me know in the comments."

"If this is your car, here's what to do next: [action]."

"Is your car on this list? Check the pinned comment."

Execution: On-screen text or voice-over at the end. Clear, short.

=== END SCRIPT ===
Metadata Synthesis Framework (Posting Package)
Every concept must produce a complete metadata package (caption, hashtags, description, CTA style, thumbnail direction) ready to paste into TikTok upload forms.

Required Metadata Fields
1. Caption (primary TikTok caption; 150 characters max for optimal preview)
Purpose: Hook the user in the feed; preview the video's core hook.

Formula: Restate the strongest hook from the script in punchy language.

Examples:

"This major recall could affect your car. Here's what to do."

"A CEO just admitted massive failures. Here's why it matters."

"100K vehicles. One defect. Are you affected?"

Requirements: Factual, grounded in source article, respects AllowedFraming.

2. Hashtag Set (5–8 hashtags)
Purpose: Increase discoverability via trending + niche tags.

Mix:

2–3 trending hashtags (trending for auto/tech/news on TikTok at posting time).

2–3 niche hashtags (specific to topic: #CarRecall, #TeslaNews, #AutoSafety, etc.).

1 brand/creator hashtag (your channel branding).

Format: Space-separated, no commas. Example: #TeslaRecall #AutoSafety #CarNews #ConsumerAlert #[YourBrandTag]

Note: Research trending hashtags at posting time; these are suggestions, not final.

3. Description (100–300 characters; TikTok shows in profile/link view)
Purpose: Provide context and SEO value for secondary viewers.

Content: Restate hook + add CTA. Example:

"Tesla announces major recall of 100K vehicles. This defect could affect your safety. Watch to learn what it means for you and what to do if you own one. Subscribe for more automotive news you need to know."

Requirements: Factual, includes full context, ends with engagement ask.

4. CTA Style (engagement driver)
Purpose: Specify which type of engagement you want (comments, saves, shares, follows).

Options:

comments – If the angle invites debate or question-response (default for controversial topics).

saves – If viewers will want to reference this later (how-to, safety tips).

shares – If the content is shareable news/public safety info.

mixed – If multiple types of engagement make sense.

Usage: Reflected in the CTA prompt in the script (e.g., "Comment if you own one" vs. "Save this if you have this car").

5. Thumbnail Guidance (for generator)
Purpose: Instruct video generation on what the thumbnail frame should emphasize.

Content: Describe the key visual that will grab attention in a feed (important for still-frame preview).

Format: 1–2 sentences. Example:

"Thumbnail should show a dramatic or surprised face reaction + on-screen text highlighting '100K vehicles' in bold red font."

"Close-up of affected vehicle component (dashboard defect) with warning label overlay."

Execution: Generators can use this guidance to select the best visual moment or add emphasis.
